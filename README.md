# React + SCSS + Vite приложение

Простое React приложение с TypeScript и SCSS модулями.

## Структура компонентов
- index.ts
- ComponentName.tsx
- ComponentName.module.scss

## Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск сервера разработки
npm run dev

# Сборка для продакшена
npm run build
```
